# penjuru-layouts

free hand-coded HTML/CSS layouts for personal sites. each one is a single HTML file (or a small handful of files) — no build step, no framework, no JavaScript framework lock-in. drop them on nekoweb / neocities / GitHub Pages / any static host and they just work.

**gallery + previews:** [layouts.penjurupikiran.com](https://layouts.penjurupikiran.com)

## using a layout

1. find one you like in the gallery
2. either:
   - **fork this repo** and trim it down to just the layout you want, *or*
   - **copy the layout's directory** out of `springtime-wiki/` (or whichever) into your own project
3. open `index.html` in your editor — most layouts have inline CSS so everything's in one place
4. swap the placeholder copy for your own
5. host it wherever — nekoweb, neocities, GitHub Pages, Cloudflare Pages, anything that serves static files
